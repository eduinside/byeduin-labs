기본 형식 (주제 지정)
아래 형식으로 [주제]에 관한 플래시카드 20장을 만들어줘. 각 줄은 반드시 다음 형식이어야 해: 앞면 내용 :: 뒷면 내용 예) 광합성 :: 식물이 빛 에너지로 포도당을 합성하는 과정 미토콘드리아 :: 세포 호흡(ATP 생성)이 일어나는 세포소기관
영단어 (단어 → 뜻)
고등학교 수준 영단어 중 [주제/파트명] 관련 단어 20개를 다음 형식으로 작성해줘. 영단어 :: 한국어 뜻 (예문 한 줄) 예) ambiguous :: 모호한 (His answer was ambiguous.)
역사·연대기 (사건 → 연도·내용)
[한국사/세계사] 중 [시대/단원] 핵심 사건 20개를 다음 형식으로 만들어줘. 사건 이름 :: 연도 — 핵심 내용 한 줄 예) 임진왜란 :: 1592년 — 일본의 조선 침략, 이순신 활약
수식·공식 (공식명 → 식)
[수학/물리/화학] [단원명] 공식 15개를 다음 형식으로 작성해줘. 공식 이름 :: 공식 (변수 설명) 예) 운동에너지 :: Ek = ½mv² (m: 질량, v: 속도)

💡 일괄 입력 탭에서 생성된 카드를 :: 구분자로 붙여넣기 하면 한번에 여러 장을 추가할 수 있습니다.

## 구현 현황

### ✅ 완료된 기능

#### 보드 관리
- [x] 여러 보드 생성·삭제
- [x] 보드 목록 홈 화면 — Flash Deck 스타일 그리드 카드 (`auto-fill, minmax(190px, 1fr)`)
- [x] 카드 썸네일: 보드 배경색 + 첫 번째 텍스트 요소 미리보기
- [x] 카드 하단: 보드 이름 / 타입 / 수정일 / 요소 수
- [x] 헤더: 공통 `.app-header` / `.app-badge` / `.app-title` / `.app-desc` 구조 적용
- [x] localStorage 자동저장 (`vives-chalkboard`)
- [x] 구형 데이터 `type` 필드 없을 때 기본값(`chalkboard`) 자동 보정

#### 보드 타입 전환
- [x] **칠판 모드**: 배경 진한 초록(`#1b4332`), 기본 글자색 흰색
- [x] **화이트보드 모드**: 배경 흰색(`#ffffff`), 기본 글자색 검정
- [x] 전환 시 기존 요소 색상 자동 변환 (흰↔검정 반전)
- [x] 다크/라이트 모드 미적용 (전용 배경색 사용)

#### 텍스트 추가
- [x] 캔버스 더블클릭으로 텍스트 추가
- [x] 기본 크기 80px, 범위 32~200px (SIZES: 32·48·64·80·96·120·160·200)
- [x] 인라인 편집 (contenteditable), Enter/Escape로 완료
- [x] 드래그로 위치 이동
- [x] 컨텍스트 메뉴: 크기 축소(A−)·확대(A+)·색상 변경·편집·삭제

#### 선 그리기
- [x] 선 모드에서 드래그로 직선 그리기
- [x] 선 드래그 이동 지원
- [x] 컨텍스트 메뉴: 색상 변경·삭제
- [x] 선 두께 기본 3px

#### 색상 팔레트 (4색)
- [x] 기본색 (칠판=흰색 `#ffffff` / 화이트보드=검정 `#111111`)
- [x] 노랑 `#fde68a`
- [x] 분홍 `#fca5a5`
- [x] 파랑 `#93c5fd`

#### 툴바 (우측 배치)
- [x] 칠판/화이트보드 전환 버튼
- [x] 텍스트 추가 버튼
- [x] 선 추가 버튼
- [x] 저장 버튼 (txt 내보내기, 자동저장 라벨 표시)
- [x] 보드 이름 입력란 테두리 스타일 추가

#### 저장 기능
- [x] 아이콘 + "저장" 텍스트 한 줄 (`btn-save-main`)
- [x] "자동저장 ●" 아래 작은 라벨 (`btn-autosave`)
- [x] 저장 버튼 → 드롭다운: TXT (텍스트만) / JPG (Canvas API로 보드 렌더링)
- [x] `saveBoard()` 호출 시 자동저장 그린 점 펄스 효과

| 항목 | 선택 |
|---|---|

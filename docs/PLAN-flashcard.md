## 핵심 기능

### 덱 관리
- 여러 덱(Deck) 생성·이름 변경·삭제
- 덱 목록 화면 (홈 뷰)
- 덱별 카드 수 + 학습 진행률 표시

### 카드 입력
- **단건 입력**: 앞면(Front) / 뒷면(Back) 텍스트 필드
- **일괄 입력**: 텍스트 블록 파싱
  - 형식: `앞면 :: 뒷면` (줄바꿈으로 구분)
  - 예시:
    ```
    photosynthesis :: 광합성
    mitosis :: 세포 분열
    ```
- 카드 편집·삭제

### 학습 모드 (Study)
- 카드 한 장씩 표시 → 클릭/스페이스로 뒤집기
- **플립 애니메이션**: CSS 3D transform (rotateY 180deg)
- 결과 버튼: ✓ 알아요 / ✗ 모르겠어요
- 진행 표시 (n / 전체)
- 완료 화면: 세션 결과 요약 (맞춤 / 틀림 / 정확도)

### 진행률 & 통계
- 카드별 마스터 여부 저장
- 덱별 전체 카드 수 / 마스터 수 / 마지막 학습일 표시
- 학습 모드 옵션: 전체 / 모르는 것만 / 섞기

## 화면 구성 (뷰)

```
view: "home"     → 덱 목록
view: "deck"     → 덱 내 카드 목록 + 편집
view: "study"    → 학습 모드 (카드 플립)
view: "result"   → 세션 결과 요약
```

### 홈 뷰 (`view: "home"`)
```
[앱 헤더]
[+ 새 덱 만들기] 버튼

[덱 카드 그리드]
┌─────────────────────┐
│ 덱 이름             │
│ 카드 12장 · 75% 완료 │
│ 마지막 학습: 오늘    │
│ [학습하기] [편집]    │
└─────────────────────┘
```

### 덱 뷰 (`view: "deck"`)
```
[← 뒤로] [덱 이름]      [학습 시작]

[일괄 입력 토글 ↕]
┌──────────────────────────────┐
│ 앞 :: 뒤 형식으로 입력...    │
│ (여러 줄 텍스트)             │
└──────────────────────────────┘
[파싱하여 추가]

[카드 목록]
┌──────────────┬──────────────┐
│ 앞면         │ 뒷면         │  [삭제]
└──────────────┴──────────────┘
```

### 학습 뷰 (`view: "study"`)
```
[← 나가기]   3 / 12   [섞기]

┌──────────────────────────────┐
│                              │
│        photosynthesis        │  ← 앞면
│                              │
│      [클릭하여 뒤집기 ▾]    │
└──────────────────────────────┘

         [뒤집으면 결과 버튼 표시]
         [ ✗ 모르겠어요 ]  [ ✓ 알아요 ]
```

## 기술 스택

| 항목 | 선택 |
|---|---|
| 구조 | 순수 HTML + CSS + Vanilla JS (빌드 없음) |
| 상태 | JS 객체 + localStorage |
| 애니메이션 | CSS 3D perspective + transition (flip) |
| 공통 스타일 | `/common/hero-theme.css`, `/common/theme.js` |
| 폰트 | JetBrains Mono (코드), Pretendard/시스템 (본문) |

React/Babel 미사용 — 단순 Vanilla JS로 뷰 전환 구현.

---

## 애니메이션 계획

| 애니메이션 | 구현 |
|---|---|
| 카드 플립 | `perspective: 1000px` + `rotateY(180deg)` + `transition: 0.4s` |
| 덱 카드 진입 | `opacity 0→1` + `translateY(8px→0)`, stagger |
| 결과 버튼 | 플립 후 `fade-in + scale(0.95→1)` |
| 완료 화면 | 진행 바 애니메이션 (`width 0→n%`) |
